## Longer Description Draft

BrowseVault helps you find, preserve, and back up your browsing history without sending it to a remote service.

Chrome's built-in history is limited and hard to search when you need to recover a page, retrace work, or export records. BrowseVault is designed as a local-first history vault: fast search, clear export, reliable backup, and precise control over your browsing memory.

The current version supports a tabbed local BrowseVault archive, background capture for new visits, manual sync from available Chrome history with per-visit expansion where Chrome exposes it, toolbar and keyboard-command reuse of an already open BrowseVault tab, Settings access to Chrome's native history page without a default-history override, advanced search syntax with site/host/domain/source/transition/visit-count/hour filters, bounded typo-tolerant keyword matching, wildcard matching, chunked local search scanning for large vaults, one-click date shortcuts, saved searches, highlighted matches, exact-day/range filters, keyboard-first live refresh, local-date grouped result scanning, Quick Open search across tabs/bookmarks/downloads/recently closed tabs/closed windows with switch/open/open-in-background/restore/copy actions, JSON backup export/import with checksum metadata, visible backup freshness/format/count/checksum status, recent activity logging for completed backup/import/export/cleanup/delete/restore/rule/reset actions, staged import preview with restore-check health and duplicate/existing/new visit counts, CSV/TSV/Google Takeout Browser History/Google My Activity/common competitor export import handling, spreadsheet-friendly CSV export with ISO/local date-time fields, HTML exports, current-result JSON/CSV/HTML export, current-result vault deletion with undo, selected-record JSON/CSV/HTML export, selected-record URL copy, selected-record batch opening with a safety cap, selected-domain blacklisting, select-visible/invert-visible/select-all-filtered bulk workflows, sticky incremental Load More and one-click Show All controls for large result sets, domain blacklist/whitelist rules, manual retention cleanup that keeps whitelisted domains, selected deletion from the vault with undo, full local vault reset, local theme/contrast/text-size/date/limit preferences, and URL-level deletion from Chrome history.
